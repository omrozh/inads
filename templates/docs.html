Super Simple Integration!

First add your domain name to the InAds system through the add domain tap from the InAds dashboard.

Place the code bellow at the bottom of your webpage's body

<script src="https://www.inads.ml/inads/<writesomethinghere>"></script>

The code above will place ads automatically. However you can customise the size, number and places of ads on your webpage as well. Use the code bellow to customize the ads. Replace <writesomething here> whatever you want. This extra part of the url is just to increase its resistance against ad block.

<div><img class=inads name=<shape>></div>

There are three different ad shapes:
  • inadstandard: width:height = 5:1
  • inadsquare: width:height = 1:1
  • inadsbanner: width:height = 1:5
By choosing one of the ad types listed above you can pick the shape of the ad you want to display and by setting the margins of the elements your can place them wherever you want. You can place as many ads as you want. If you are giving additional styling to ads always give the styling to the parent div.
Important Warning: Always place the script element inside your body tags and bellow the img elements

How to add a domain?

Make sure to download the inadsconfirm.txt file from domain add page of your dashboard and add it to your website as inadsconfirm.txt without any change. The file must be accessible through http://<yourdomain>/inadsconfirm.txt

Than add the domain to the database by filing the form. Your domain will be confirmed if the inadsconfirm.txt file can be detected in your website and you will be ready to monetize your website.
