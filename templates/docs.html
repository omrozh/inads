Super Simple Integration!

First add your domain name to the InAds system through the add domain tap from the InAds dashboard.

Place the code bellow at the bottom of your webpage's body

<script src="https://www.inadsglobal.com/inads/<writesomethinghere>"></script>

The code above will place ads automatically. However you can customise the size, number and places of ads on your webpage as well. Use the code bellow to customize the ads. Replace <writesomething here> whatever you want. This extra part of the url is just to increase its resistance against ad block.

<div><img class=inads name=<shape>></div>

There are four different ad types:
  • inadstandard: width:height = 5:1
  • inadsquare: width:height = 1:1
  • inadsbanner: width:height = 1:5
  • inadsvideo: for videos instead of <div><img class=inads name=<shape>></div> use <div><video class=inads name=<shape> autoplay loop muted playsinline controls></div>
By choosing one of the ad types listed above you can pick the shape of the ad you want to display and by setting the margins of the elements your can place them wherever you want. You can place as many ads as you want. If you are giving additional styling to ads always give the styling to the parent div.
Important Warning: Always place the script element inside your body tags and bellow the img elements

How to add a domain?

Make sure to download the inadsconfirm.txt file from domain add page of your dashboard and add it to your website as inadsconfirm.txt without any change. The file must be accessible through http://<yourdomain>/inadsconfirm.txt

Than add the domain to the database by filing the form. Your domain will be confirmed if the inadsconfirm.txt file can be detected in your website and you will be ready to monetize your website.

InAds with Unity

How to use InAds with Unity

Go to add domain tab and enter unity, with lowercase letters, as your domain and get your API key from the website performance tab.

Download the InAds Script for Unity. Create a Raw Image object in your UI canvas and add the InAds Script as a component.

Kolay Entegrasyon!

Öncelikle alan adınızı InAds gösterge panelinin alan adı ekleme sekmesinden InAds sistemine ekleyin.

Aşağıda yer alan kodu web sayfanıza ekleyin.

<script src="https://www.inadsglobal.com/inads/<writesomethinghere>"></script>

Eğer sadece bu kodu yerleştirip bırakırsanız otomatik yerleştirmeyi etkinleştirmiş olursunuz verilen script sizin yerinize web sayfanıza reklamları ekleyecektir. Fakat eğer isterseniz yukarıdaki kodu kullanarak reklamlarınızın sayısını, boyutlarını ve yerlerini özelleştirebilirsiniz.

<div><img class=inads name=<shape>></div>

Şu anlık web sayfanıza ekleyebileceğiniz reklam çeşitleri aşağıdaki gibidir.
  • inadstandard: width:height = 5:1
  • inadsquare: width:height = 1:1
  • inadsbanner: width:height = 1:5
  • inadsvideo: Videolarda bu kodun yerine <div><img class=inads name=<shape>></div> bu kodu kullanın <div><video class=inads name=<shape> autoplay loop muted playsinline controls></div>
Yukarıdaki listelenmiş reklam tiplerinden birini seçerek reklamlarınızın şeklini seçebilirsiniz. Eğer bu reklamlara ekstra stil vermek isterseniz bunu div üst öğesi üzerinden yapmanızı öneririz.
Önemli uyarı: En iyi sonuçları almak için reklam scriptini sayfanızın en altına yerleştirmenizi tavsiye ederiz.

Nasıl yeni bir alan adı eklerim?

Öncelikle alan adı ekleme sekmesinden inadsconfirm.txt dosyasını indirin, websitenize ekleyin ve dosyanın http://<alan adınız>/inadsconfirm.txt adresinden erişilebilir olduğuna emin olun.

Sonrasında sayfada yer alan formu doldurarak işlemi tamamlayın ve kullanmaya başlayın.

Unity ile InAds

InAds Unity ile nasıl kullanılır?

Alan adı ekleme sekmesine gidin ve alan adı olarak küçük harfler ile unity yazın. Sistem size kullanmanız için bir public key atayacak bunu web sitesi performans sekmesinden veya hesap sekmesinden görüntüleyebilirsiniz.

InAds Script for Unity dosyasını indirin. UI'ınızda bir Raw Image objesi oluşturun ve InAds Scriptini bu objeye bileşen olarak ekleyin.
